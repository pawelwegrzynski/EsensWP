== Description ==

[Premium](https://tishonator.com/plugins/showcase-your-team) | [Demo](https://tishonator.com/plugins-demo/showcase-your-team/)

Add Your Team Members (Name, Photo, Bio, Social Links) from Admin Panel and Showcase Your Team with Mobile-Friendly and Responsive Grid with Fancy Hover Effect by Inserting a Widget, Shorcode or a Gutenberg Block.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `showcase-your-team` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Admin Panel -> Left Menu -> Team Members -> Add New
4. Create and Publish Team Members
5. Insert the shortcode [team-members count="NUMBER"] to display Team Members to any page or post
6. Insert 'Team Members' Gutenberg block in your page or post to display Team Members
7. Insert 'Team Members' Widget if you want to showcase your team as a widget
5. Enjoy!
